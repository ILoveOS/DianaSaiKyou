# Diana最強

## 环境
```
node.js vue
```

## 安装依赖
```
npm install
```

## 编译
```
npm run build
```

## 发布release

### 大版本更新
```
npm run major
```

### 小版本更新
```
npm run minor
```

### 补丁更新
```
npm run patch
```

## 安装
```
浏览器打开chrome://extension,启用开发人员模式->加载已解压的拓展->选择npm run build后生成的dist文件夹
```

## 版本管理配置
修改文件[repo.json](/public/repo.json)

```
owner->GitHub用户名
repo->项目的仓库名称
url->项目的github地址
version->项目版本 
```
